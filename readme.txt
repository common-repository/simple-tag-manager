Easily and automatically add Google Tag Manager and Google Analytics in your website.

== Description ==

Simple Tag Manager is a simple plugin created in order to easily install Google Tag Manager and Google Analytics in your WordPress website.
Just copy and paste the property code or the tracking ID and ... it's done!

== Installation ==

Installing this plugin is very simple:

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Simple Tag Manager screen to configure the plugin

== Frequently Asked Questions ==

= Why isn’t the output displaying correctly? =
Two possibilities: First, you haven’t yet specified the ID in the admin panel, or second, your theme is missing a <?php wp_footer(); ?> call.
